Identification of Bytes with the Elements of GF(2^8)

Rijndael暗号では、「byte バイト」が重要な役割を果たします。バイトは、16進数のペアで書かれています。
RijndaelはAESの暗号アルゴリズムに採用された共通鍵（秘密鍵）暗号方式です。

例

16進数の2F{2F}2Fのペアは、長さ4のビットベクトルの001011110010 111100101111のペアに対応しています。つまり、このペアはバイト「001011110010111100101111」を表します。

16進数のペアA1{A1}A1は、ビットベクトルのペア101000011010 000110100001に対応しています。つまり、そのペアはバイト101000011010000110100001を表します。

SubBytes

SubBytes (state) 非線形関数です。stateの個々のバイトを変換します。この変換をS-Boxと呼びます。 stateの各バイトは、次のようにマッピングされます：
b←Ab−1⊕c\boldsymbol{b} \leftarrow \boldsymbol{A} \boldsymbol{b}^{-1} \oplus \boldsymbol{c} \quadb←Ab−1⊕c ( ⊕\oplus⊕ N進サークルプラス演算子)

ここのbはビットベクターとして、可能な引数は28個しかないので、S-boxを表にすることができます。そして、SubBytesはテーブルルックアップ(table lookups )で実装できます。
